## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone or download this repository
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

## Usage

1. **Choose a Template**: Select from Modern, Classic, or Minimal templates
2. **Fill in Your Information**:
   - Personal Information (name, email, phone, etc.)
   - Professional Summary
   - Work Experience (add multiple entries)
   - Education (add multiple entries)
   - Skills (with proficiency levels)
3. **Preview**: See your resume update in real-time on the right side
4. **Download**: Click the "Download PDF" button to save your resume

## Templates

- **Modern**: Colorful design with gradient accents and skill bars
- **Classic**: Traditional black and white format, perfect for formal applications
- **Minimal**: Clean, minimalist design with subtle colors
